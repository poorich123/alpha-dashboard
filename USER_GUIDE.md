# 📖 คู่มือใช้งาน Alpha Dashboard (ฉบับเข้าใจง่าย)

> เครื่องมือนี้ช่วยตอบ 4 คำถามหลักของนักลงทุน:
> **1) หุ้นตัวนี้ถูกหรือแพง?** · **2) ร้อนแรงเกินไป (ฟองสบู่) ไหม?** ·
> **3) เงินใหญ่ (กองเก่ง) กำลังทำอะไร?** · **4) ตอนนี้ตลาดอยู่ในภาวะไหน ควรถืออะไร?**
>
> ⚠️ ทุกตัวเลขเป็น **เครื่องมือช่วยตัดสินใจ ไม่ใช่คำแนะนำลงทุน** — ใช้เป็นกรอบคิด ไม่ใช่ปุ่มซื้อ/ขาย

---

## 🗺️ แผนผังหน้าใช้งานบ่อย

| อยากรู้ว่า… | ไปหน้า | ดูอะไร |
|---|---|---|
| หุ้นตัวนี้ควรซื้อไหม ราคาเหมาะหรือยัง | **Analyzer** | Signal + Fair Value + Bubble Score |
| ตอนนี้ตลาดเสี่ยงอะไร ควรถืออะไร | **Macro Risk** | Dalio Regime + ปัจจัยเสี่ยง 7 ตัว |
| กองเก่งซื้อ/ขายอะไร | **Smart Money** | Conviction Fund Tracker |

| สแกนหาหุ้นเด่นทั้งตลาด | **Market** | TP24-style screener |

## 1️⃣ Analyzer — วิเคราะห์หุ้นรายตัว

พิมพ์ ticker (เช่น `NVDA`) แล้วกด Analyze จะได้ 3 ส่วนสำคัญ:

### 🎯 Signal + คะแนน (ด้านบน)
- **Signal:** STRONG BUY → STRONG SELL (จากเทคนิคัล 8 ข้อ)
- **TP1/TP2/TP3 + SL + โซนสะสม:** เป้าทำกำไร / จุดตัดขาดทุน / ช่วงราคาที่ควรเก็บ
- **Entry Strategy:** บอกว่าควรเข้า "เต็มไม้ / ครึ่งไม้ / รอย่อ" ตามราคาปัจจุบัน

### 💰 Fair Value · Margin of Safety (ของใหม่ — มุมมองมูลค่า)
ตอบว่า **"ราคาตอนนี้ถูกหรือแพงเทียบมูลค่าที่แท้จริง"** ด้วย **5 วิธีแบบที่นักวิเคราะห์ Wall St ใช้** ถ่วงน้ำหนักรวมกัน (วิธีไหนข้อมูลไม่พอจะข้ามแล้วเฉลี่ยใหม่):

| วิธี | คิดจาก | น้ำหนัก | ใช้กับ |
|---|---|---|---|
| **Analyst** | เป้าเฉลี่ยนักวิเคราะห์ 12 เดือน (Street consensus) | 35% | ทุกตัวที่มี coverage |
| **DCF** | กระแสเงินสดอนาคต 10 ปี | 25% | บริษัททำเงินสดได้ (FCF บวก) |
| **Comparable** | EPS × fair P/E (base 11× + growth premium, 10–38×) | 20% | บริษัทมีกำไร |
| **EV/Sales** | รายได้ × P/S (1–12× ตาม growth) − net debt | 12% | หุ้น growth/ยังไม่กำไร |
| **Asset** | book value / Graham Number | 8%* | เป็นพื้น (floor) |

\* Asset ลดเหลือครึ่งถ้าบริษัทขาดทุน · DCF/Comparable จะข้ามอัตโนมัติถ้า FCF/EPS ติดลบ แล้วโมเดลพึ่ง EV/Sales + Analyst แทน (แบบที่นักวิเคราะห์ทำกับหุ้น pre-profit)

**อ่านตัวเลขสำคัญ — Margin of Safety (MoS):**
- 🟢 **บวกเยอะ (+30% ขึ้น) = Deep Value** → ราคาต่ำกว่ามูลค่ามาก (น่าสนใจ)
- 🟢 +10–30% = Undervalued · 🟡 ±10% = Fair Value (พอดี)
- 🟠 −10 ถึง −25% = Overvalued · 🔴 ต่ำกว่า −25% = **Expensive** (แพง)

> 💡 **ตัวอย่าง:** MoS = −49% แปลว่าราคาแพงกว่ามูลค่าที่ประเมิน ~49% → ระวังการไล่ราคา
> ETF (เช่น SPY) จะขึ้น "Valuation N/A" เพราะโมเดลนี้ใช้กับหุ้นรายตัวเท่านั้น

### 🌊 Bubble Score · Dalio 6-Point (ของใหม่ — มุมมองความร้อนแรง)
ตอบว่า **"หุ้นนี้ฟองสบู่/ร้อนแรงเกินไปไหม"** ด้วยเกจ 6 ด้าน (คะแนนสูง = เสี่ยงไล่ราคา):

1. **มูลค่าแพง** — ราคาเทียบมูลค่าพื้นฐาน
2. **คาดหวังโตเกินจริง** — PEG (ราคาสะท้อนการเติบโตมากไป?)
3. **ราคาวิ่งไกลจากเทรนด์** — สูงกว่าเส้นค่าเฉลี่ย 200 วัน / ใกล้จุดสูงสุด 52 สัปดาห์แค่ไหน
4. **แรงซื้อร้อนแรง** — RSI + Bollinger (ซื้อมากเกินไป?)
5. **ความผันผวน/เลเวอเรจ** — β + ความผันผวน (proxy การเก็งกำไร)
6. **เร่งตัวพาราโบลา** — โมเมนตัมสั้นพุ่งแรงกว่าระยะยาว + วอลุ่มพุ่ง

**อ่านคะแนนรวม:** ต่ำ (เขียว) → Moderate → Frothy (เหลือง) → Elevated (ส้ม) → **Bubble Risk (แดง)**

> 💡 **วิธีใช้คู่กัน:** หุ้น Signal = BUY แต่ Bubble Score สูง + MoS แพง → อาจ "ดีแต่แพง" ควรรอย่อ
> หุ้น MoS ถูก + Bubble ต่ำ + Signal ขึ้น = setup ที่สวยที่สุด

## 2️⃣ Smart Money — เงินใหญ่ทำอะไร

### 🎯 Conviction Fund Tracker (ของใหม่ — แทน 13F แบบเดิม)
ตาม **5 กองเก่ง** ว่าซื้อ/ตัดอะไรในไตรมาสล่าสุด (จาก 13F ของ SEC):

🦅 Druckenmiller · 🎯 Ackman · 🐯 Tiger Global · 🌊 Coatue · 🤖 Aschenbrenner (กอง AI)

กดเลือกกอง → เห็นตารางหุ้นพร้อมป้ายสี:
- 🟢 **NEW** = เปิดสถานะใหม่ (conviction สูงสุด — เพิ่งเชื่อ) · **ADD** = ซื้อเพิ่ม
- 🔴 **TRIM** = ลดสัดส่วน · **EXIT** = ขายทิ้งหมด · HOLD = คงเดิม
- คอลัมน์ **Δ Value / Δ Shares** = เปลี่ยนแปลงเทียบไตรมาสก่อน

> 💡 **ตัวอย่างจริง:** Ackman เปิด **NEW Microsoft +$2.09B** และ **EXIT Hilton** ในไตรมาสล่าสุด
> 🔵 **แถวไฮไลต์ฟ้า + ป้าย "ในพอร์ต"** = หุ้นที่คุณถืออยู่ตรงกับกอง → ดูว่าเขากำลังเพิ่มหรือลด

**ข้อจำกัดที่ควรรู้:** 13F ล่าช้า 45 วัน · แสดงเฉพาะหุ้น long ในสหรัฐ (ไม่รวม short/ออปชัน/ต่างประเทศ)

### อีก 2 พาเนลในหน้าเดียวกัน
- **CFTC COT** = hedge fund ถือ long/short ในฟิวเจอร์สมากไปไหม (z-score >2 = สุดโต่ง → สัญญาณสวนทาง)
- **Dealer GEX** = แนวรับ/แนวต้านจากออปชัน (zero-gamma = จุดพลิก)

## 3️⃣ Analytics — Correlation Matrix (ของใหม่)

ตอบว่า **"พอร์ตเรากระจายความเสี่ยงจริง หรือหุ้นวิ่งตามกันหมด"**

เลื่อนลงล่างสุดของหน้า Analytics → ตารางสี (heatmap):
- 🔴 **แดง = วิ่งไปทางเดียวกัน** (ถ้าตกก็ตกพร้อมกัน — เสี่ยงกระจุก)
- ⬜ เทา = ไม่สัมพันธ์กัน (กระจายดี) · 🔵 น้ำเงิน = วิ่งสวนกัน (ช่วย hedge)

**ปุ่ม toggle Sector ↔ Category** = จัดกลุ่มตามเซกเตอร์ หรือตามหมวด (core/satellite/spec)

อ่านอะไร:
- **ค่าเฉลี่ย correlation ทั้งพอร์ต** สูง (>0.6) = กระจายน้อยกว่าที่คิด
- **correlation เฉลี่ยรายกลุ่ม** = กลุ่มไหนกระจุก
- **เตือนคู่ ≥0.80** = คู่หุ้นที่แทบเป็นตัวเดียวกัน

> 💡 **ตัวอย่าง:** ถือทั้ง XOM และ CVX → corr 0.85 (น้ำมัน 2 ตัว) = จริง ๆ เหมือนถือก้อนเดียว
> ถ้าเห็นแบบนี้เยอะ ควรกระจายไปเซกเตอร์อื่นที่ corr ต่ำ/ติดลบ

## 4️⃣ Macro Risk — Dalio Regime Detector (ของใหม่)

ตอบว่า **"ตอนนี้เศรษฐกิจอยู่โหมดไหน ควรถือสินทรัพย์แบบใด"**

ใช้กรอบ Ray Dalio: ดู 2 แกน — **การเติบโต (Growth)** กับ **เงินเฟ้อ (Inflation)** ว่ากำลังขึ้นหรือลง → ได้ 4 ภาวะ:

| ภาวะ | สภาพ | ของที่ได้เปรียบ | ของที่ควรเลี่ยง |
|---|---|---|---|
| 🌤️ **Goldilocks** | โต↑ เฟ้อ↓ | หุ้นเติบโต/เทค, credit | เงินสด, สินค้าโภคภัณฑ์ |
| 🔥 **Reflation** | โต↑ เฟ้อ↑ | value, พลังงาน, commodities, TIPS | พันธบัตรยาว, หุ้น P/E สูง |
| 🥵 **Stagflation** | โต↓ เฟ้อ↑ | ทองคำ, commodities, defensive, เงินสด | หุ้นเติบโต, cyclicals |
| ❄️ **Deflation** | โต↓ เฟ้อ↓ | พันธบัตรยาว (TLT), quality, เงินสด | cyclicals, high-beta, commodities |

ในหน้าจะเห็น **ตาราง 2×2 ไฮไลต์ช่องที่เราอยู่** + แถบความมั่นใจของแต่ละแกน + รายการสัญญาณที่ใช้ตัดสิน (copper/gold, SPY, credit, oil, yields)

> 💡 **ตัวอย่าง:** ถ้าระบบขึ้น **Reflation** → เอียงไปพลังงาน/value, ระวังหุ้นเทค P/E สูงและพันธบัตรยาว
> ด้านล่างยังมีปัจจัยเสี่ยงรายตัว 7 ตัว (น้ำมัน, ทอง, yields, DXY, BTC, bond vol, yield curve) พร้อม "What to do"

## 🔄 Workflow แนะนำ (รวมทุกเครื่องมือ)

1. **เช็คภาพใหญ่ก่อน** → หน้า **Macro Risk**: ดู Dalio Regime ว่าควรเอียงไปสินทรัพย์แบบไหน + มีความเสี่ยงเฉียบพลันไหม
2. **หาไอเดีย** → หน้า **Market** (สแกน) หรือดู **Smart Money** ว่ากองเก่งเพิ่งซื้ออะไร (NEW/ADD)
3. **เจาะรายตัว** → หน้า **Analyzer**: ดู Signal + **Fair Value (ถูก/แพง)** + **Bubble Score (ร้อนไป?)** พร้อมกัน
4. **ก่อนเข้าซื้อ** → เช็ค **Correlation Matrix** ว่าหุ้นใหม่จะทำให้พอร์ตกระจุกตัวขึ้นไหม
5. **ตัดสินใจ** → setup ที่ดี = ภาวะตลาดเอื้อ + กองเก่งสนใจ + ราคาไม่แพง (MoS ดี) + ไม่ร้อนเกิน (Bubble ต่ำ) + ไม่ทำพอร์ตกระจุก

## ⚠️ ข้อควรระวัง

- ทั้งหมดเป็น **เครื่องมือช่วยคิด ไม่ใช่คำแนะนำลงทุน** — ตัดสินใจเองเสมอ
- **Fair Value / Bubble** อ่อนไหวต่อสมมติฐานมาก (growth, discount rate) — ใช้เป็นกรอบ
- **Conviction Tracker** ดูจาก 13F ที่ล่าช้า 45 วัน — กองอาจขายไปแล้วก็ได้
- **Dalio Regime** ใช้ proxy จากตลาด (ไม่ใช่ GDP/CPI จริง) — เปลี่ยนได้เร็ว
- ข้อมูลดึงจาก Yahoo/SEC/CFTC ฟรี — บางครั้งอาจช้า/ขาด ลองกด Refresh
